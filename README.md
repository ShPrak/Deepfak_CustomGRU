# Deepfak_CustomGRU
The public code base for the research paper-Attention-Augmented Hybrid Framework with Evolutionary Optimization for Robust Deepfake Detection

## Overview
This code contains the core implementation of the modified GRU architecture used in our experiments. It is shared to support understanding and reproducibility of the methodological contribution presented in the paper.

## Datasets

The datasets used in the original study, including FaceForensics++ and Celeb-DF v2, are not included in this repository.

These datasets are subject to usage restrictions and must be obtained directly from their respective authors or official sources.<br>
FaceForensics++ https://github.com/ondyari/faceforensics <br>
Celeb-DF V2 https://github.com/yuezunli/celeb-deepfakeforensics
